# NusaRasa

NusaRasa - Aplikasi mobile yang menghadirkan ragam resep makanan khas dari setiap pulau di Indonesia. Dari Sumatera hingga Papua, kami siap membawa kelezatan tradisional langsung ke dapur Anda!
